Описание проекта.
Простой слайдер с небольшим описанием слайда.
Написан на чистом JS ,без использования jQuery и прочих библиотек.
Не кликабельный. 
Необязательные аттрибуты в блоке div класса 'block' 
visibleTime - время видимости одного слайда (3000-50000) по умолчанию 3000
blockWidth - ширина свернутого блока (0-30) по умолчанию 15

класс '_img' для картинок 
класс '_txt' для параграфа с описанием
класс 'fullblock' весь блок
класс 'block' блок с изображениями
класс 'textblock' блок для текста
программно добавляется текст 

код 
<div class="textblock">
            <div id="txtID" class="text"></div>
        </div>
после блока div с классом 'block' должен быть как есть
